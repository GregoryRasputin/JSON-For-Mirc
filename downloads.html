Stable
v0.2.41 - 12th June 2016
Requirements

Mirc: 7.32+ or AdiIRC 2.2+

MS Windows 2000 or later

If you are using AdiIRC 64bit, you will need to install the tsc.dll for this script to work

Installation

Download JSONForMirc v0.2.41.mrc

Move the file to a directory of your choice

In mIRC/AdiIRC enter the following into an input box and hit enter:
//load -rs $qt($$sfile($mircdir, Load JSONFormIRC, Load))

In the box that appears, navigate to where you saved the JSONForMirc v0.2.41.mrc file

Select it then click "Load"
