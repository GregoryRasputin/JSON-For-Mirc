Stable
v0.2.41
Requirements
  • MS Windows 2000 or later
  • mIRC v7.32+ or AdiIRC v2.2+
  • For 64bit AdiIRC versions, the tsc.dll will need to be installed
Installation
  • Download JSONForMirc v0.2.41.mrc
  • Move the file to a directory of your choice
  • In mIRC/AdiIRC enter the following into an input box and hit enter://load -rs $qt($$sfile($mircdir, Load JSONFormIRC, Load))
  • Select the JSONForMirc v0.2.41.mrc file
  • Click "Load"
