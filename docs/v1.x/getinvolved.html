Feedback
If you have a suggestion or believe you've encountered a bug feel free to open an issue ticket on the project's github. Suggestions are always welcome and bugs are a top priority; as such the issue tracker is checked daily and all open tickets are addressed immediately.
Without user feedback there can be no developer progression
