Installation

After uninstalling all previous versions of JSONForMirc:

  1. Download the desired version
  2. Move the file to the directory of your choosing
  3. Within mIRC enter the following: //load -rs $qt($$sfile($mircdir, Load JSONForMirc, Load))
  4. In the window that appears navigate to where you stored JSONForMirc.mrc
  5. Select JSONForMirc.mrc
  6. Click the Load Button
